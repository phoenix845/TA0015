### Environment Requirements
To run this project, the following environment variables are required:
- `MONGO_URI`: MongoDB connection string.
- `JWT_SECRET` / `JWT_EXPIRE`: For signing authentication tokens.
- `OPENWEATHER_KEY`: For fetching meteorological data.
- Cloudinary credentials (if image uploads are utilized).

### Running Locally
1. **Backend:** Navigate to `/backend`, run `npm install`, then start the development server using `npm run dev` (runs on port 5000).
2. **Frontend:** Navigate to `/frontend`, run `npm install`, then start the react app using `npm start` (runs on port 3000).
